# Τηλεφωνικός κατάλογος

Μικρή εφαρμογή Blazor Server: βλέπεις τους άλλους χρήστες και κοινές επαφές που προσθέτει ο καθένας. Τα δεδομένα μένουν σε SQL Server — για τοπικά tests χρησιμοποιώ LocalDB.

Χρειάζεσαι .NET 9 και LocalDB (συνήθως μαζί με Visual Studio). Αντιγράφεις το repo, μπαίνεις στο φάκελο `phonemanagement` και τρέχεις `dotnet run`. Στην πρώτη εκκίνηση φτιάχνεται/ενημερώνεται η βάση μόνη της. Άνοιξε το URL που εμφανίζεται στην κονσόλα.

Για να δεις κατάλογο πρέπει να συνδεθείς. Admin: `admin@test.com` / `admin` (ή γράψε μόνο `admin` στο πεδίο σύνδεσης). Αν είχες παλιά `admin@admin.com` στη βάση, στο επόμενο τρέξιμο της εφαρμογής το seed το περνάει αυτόματα στο `admin@test.com` με κωδικό `admin`. Οι υπόλοιποι δοκιμαστικοί χρήστες έχουν `123456`.
